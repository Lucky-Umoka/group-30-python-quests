# Group 30 Python Quests

A repository for tracking and completing Python coding quests.

## How to Run a Quest

From the main directory, run any quest using:
- bash
python3 quest_XX_name.py

## Levels
- Level 1 (Quests 1-5): Variables and Printing
- Level 2 (Quests 6-10): User Input and Math
- Level 3 (Quests 11-15): Conditional Statements
- Level 4 (Quests 16-20): Loops
- Level 5 (Quests 21-24): Functions
- Level 6 (Quests 25-30): Grand Challenge
